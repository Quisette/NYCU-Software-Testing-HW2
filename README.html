Implementation

class Calc {
    static add(a, b) {
        return a + b;
    }

    static subtract(a, b) {
        return a - b;
    }

    static multiply(a, b) {
        return a * b;
    }

    static divide(a, b) {
        if (b === 0) {
            throw new Error('Division by zero is not allowed');
        }
        return a / b;
    }
}

Features:

  • 22 lines of clean, minimal code
  • Full support for integers and floating-point numbers
  • Error handling for division by zero
